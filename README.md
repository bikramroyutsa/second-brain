# Second Brain Note Taking App

## Features
+ Take, Edit, Delete notes
+ Full Text Search
+ Semantic Search
+ Chatbot with knowledge base
+ Note Linking and Map View
+ Auto generated Flashcards

## Stack
+ NextJS
+ Supabase (Postgres DB, Edge functions, Authentication)
+ React Flow (For Map View) with d3-force (for layouting)

## Models
+ supabase/gte-small for embedding
+ llama3.2:3b for Chatbot
